## How This Document Was Created

**Process:** Systematic credibility evaluation of all 25 source documents using standardized criteria
**Source Documents:** All primary sources (PDFs), synthesized analyses, competitor research, case studies
**Date Generated:** 2026-02-10

During the research session, the AI agent evaluated each source across 5 dimensions: external validation (25%), recency (20%), methodology transparency (20%), bias assessment (20%), and corroboration (15%). Sources were assigned tier ratings (TIER 1-5) to inform evidence weighting throughout the comprehensive report.

---

## Credibility Assessment Framework

### Evaluation Dimensions

| Dimension | Weight | Description |
|-----------|--------|-------------|
| **External Validation** | 25% | Can claims be verified outside the source? |
| **Recency** | 20% | How current is the information? |
| **Methodology Transparency** | 20% | Is data collection method explained? |
| **Bias Assessment** | 20% | What incentives might skew the data? |
| **Corroboration** | 15% | Do other sources confirm this? |

### Credibility Ratings

| Rating | Score | Meaning |
|--------|-------|---------|
| **TIER 1** | 90-100 | Gold standard — externally verifiable, transparent methodology |
| **TIER 2** | 75-89 | Reliable — good sourcing, minor gaps |
| **TIER 3** | 60-74 | Usable with caveats — internal data, limited verification |
| **TIER 4** | 40-59 | Use cautiously — significant methodology or bias concerns |
| **TIER 5** | 0-39 | Insufficient — unverifiable claims, high bias risk |

## Recommendations for Report

### Citing High-Credibility Claims
```
Format: "[Claim] [STAT-ID; TIER X confidence]"
Example: "Blinkist maintains a 4.7-star app rating [STAT-003; TIER 1]"
```

### Citing Medium-Credibility Claims
```
Format: "[Claim] [STAT-ID; TIER X] — [caveat]"
Example: "68% Month 1 adoption [STAT-008; TIER 3] — methodology not disclosed"
```

### Flagging Low-Credibility Claims
```
Format: "⚠️ [Claim] [STAT-ID; TIER 4-5] — insufficient evidence"
Example: "⚠️ 3x engagement vs traditional LMS [STAT-007; TIER 5] — benchmark source unknown"
```
