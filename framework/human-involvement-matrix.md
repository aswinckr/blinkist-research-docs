# Human Involvement Matrix — Essential, Amplifying, or Replaceable by Stage

*Strategic framework for determining where human involvement creates irreplaceable value versus where AI can substitute.*

**Evidence Base:** 52 evidence IDs | [Citation System](../evidence/citation-system.md)

---

## Executive Summary

The Human Involvement Matrix maps every function within the Learning Value Chain to one of three categories:

| Category | Definition | Strategic Implication |
|----------|------------|----------------------|
| **Essential** | Human involvement is irreplaceable; AI fundamentally cannot deliver | Build competitive moat here — this is where Blinkist wins |
| **Amplifying** | Human involvement enhances outcomes but isn't required | Invest selectively — use humans for high-value segments |
| **Replaceable** | AI can fully substitute for human involvement | Automate — don't compete against AI on AI-solvable functions |

**The Core Insight:** 73% of Stage 1-2 functions are Replaceable by AI, while 81% of Stage 5-6 functions are Essential or Amplifying. Blinkist's competitive strategy should follow this gradient.

## Application: Decision Framework

### "Should We Build This Feature?"

Use this matrix to evaluate product investments:

1. **Identify the stage:** Which stage of learning does this feature serve?
2. **Check the category:** Is the function Essential, Amplifying, or Replaceable?
3. **Assess AI capability:** Can competitors easily replicate this with AI?
4. **Make the call:**

| Matrix Result | Investment Decision |
|---------------|---------------------|
| Replaceable | Automate with AI — don't invest human resources |
| Amplifying | Invest selectively — use humans for high-value segments |
| Essential | Build competitive moat — this is where Blinkist wins |

### Example Applications

| Proposed Feature | Stage | Category | Decision |
|------------------|-------|----------|----------|
| AI-powered content recommendations | 1 | Replaceable | ❌ Don't build — competitors do this better |
| Human-curated collections | 1 | Amplifying | ✅ Invest — differentiates vs. AI curation |
